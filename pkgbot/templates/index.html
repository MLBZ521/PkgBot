PkgBot: Systems Online

PkgBot will record software version history for Jamf Pro.

PkgBot is a framework to manage the lifecycle of software packaging, testing, and then promotion from development to production environments. It utilizes the open source project AutoPkg to download and package software and a Slack Bot is utilized to send notifications and receive commands.

Within this site, you will find software version history including it's current status, as well as details on the available AutoPkg recipes.

This site will provide greater visibility into third party software updates.
